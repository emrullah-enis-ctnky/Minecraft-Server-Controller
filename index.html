⚙️ Systemd Auto-Start Service Configuration Guide

To run this controller automatically on boot as a background service on CachyOS, follow these commands:

# 1. Copy the service file to the systemd directory:
sudo cp mcs-controller.service /etc/systemd/system/mcs-controller.service

# 2. Reload systemd files:
sudo systemctl daemon-reload

# 3. Enable the service to run on boot:
sudo systemctl enable mcs-controller.service

# 4. Start the service immediately:
sudo systemctl start mcs-controller.service

# 5. Check status of the controller:
sudo systemctl status mcs-controller.service
